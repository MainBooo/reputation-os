# ReputationOS — питч-дек для Telderi

## Что это

**ReputationOS** — SaaS-платформа для управления репутацией бизнеса в интернете.
Агрегирует отзывы, рейтинги и упоминания из Яндекс Карт, 2ГИС, веб-поиска и Telegram,
показывает аналитику и помогает отвечать на отзывы с помощью ИИ.

Целевая аудитория: малый и средний бизнес, маркетинговые и репутационные агентства.

## Интеграции

- **Яндекс Карты** — парсинг отзывов и рейтинга
- **2ГИС** — парсинг отзывов и рейтинга
- **Web** — мониторинг упоминаний через поисковый индекс (DuckDuckGo / Яндекс)
- **PageWatch** — отслеживание изменений на конкретных страницах
- **Telegram** — уведомления о новых отзывах и упоминаниях
- **YandexGPT** — генерация ответов на отзывы

Модель: подписка (recurring), оплата через ЮKassa. Биллинг полностью реализован:
checkout, webhook, idempotent-обработка, автоактивация подписки.

---

## Технический стек

**Backend:** NestJS (TypeScript), PostgreSQL, Prisma ORM, BullMQ (Redis), Playwright
**Frontend:** Next.js 14 (App Router), TypeScript
**Инфраструктура:** PM2, Docker (PostgreSQL + Redis), pnpm workspaces (monorepo)
**AI:** YandexGPT API
**Платёжная система:** ЮKassa (реализована) + Mock-провайдер для тестирования

---

## Архитектура

```
apps/
  api/      — NestJS REST API (14 модулей: auth, billing, companies, analytics, ...)
  frontend/ — Next.js клиент
  worker/   — BullMQ воркер (sync отзывов, рейтингов, упоминаний, page-watch)
  landing/  — лендинг
  bot/      — Telegram-бот
```

Все сервисы работают через PM2, stateless, горизонтально масштабируемы.

## Что уже работает

- Регистрация / авторизация (JWT, воркспейсы, роли: OWNER / ADMIN / MEMBER)

- AI-ответы на отзывы через YandexGPT

- Полный биллинг (планы, checkout, webhook, entitlements, лимиты)
- Unit-тесты на критические пути

## Почему продаю

Проект реализован как MVP для валидации ниши управления репутацией в России.
Ниша подтверждена — есть пользователи, парсинг работает в продакшне.
Продаю, так как переключаюсь на другое направление и не готов масштабировать
этот продукт самостоятельно. Покупатель получает готовый работающий SaaS
с кодом, базой данных, инфраструктурой и первыми пользователями.
